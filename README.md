# BidTabsData

Shared versioned dataset of BidTabs .xls files distributed via GitHub Releases for consuming repos.

## Overview

This repository serves as a centralized location for versioned BidTabs data files. The data is distributed through GitHub Releases, allowing consuming repositories to reliably fetch specific versions of the dataset.

## Structure

- **data/**: Contains the BidTabs .xls data files
- **tools/**: Contains utilities and scripts for working with the BidTabs data

## Usage

Consuming repositories can download specific versions of the data from the GitHub Releases page.
